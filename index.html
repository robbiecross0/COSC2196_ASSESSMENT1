Project

Overview

Developing a mobile-based social media application that allows connection with like-minded individuals to participate in fitness-based activities. With the threat of gyms closing, local sports competitions shut-down and even people looking to share content and at-home workouts there will be a severe lack of options available for people to continue to exercise and participate in group fitness. The application would be easy to use, non-intrusive from a privacy perspective and allow users to tailor what they see based on activities, exercise, distance and gender. Conforming to government guidelines regarding social distancing will be a requirement, as such outdoor activities such as running, swimming, or limiting interaction/number of people would be a key feature. Using Java to develop this application for smartphones would provide a flexible and familiar UI, and being the official language for Android would ensure there is sufficient resources and support available for any issues encountered.

Motivation

The social impact of the covid-19 virus means that people's lifestyle regarding exercise has the potential to be severely impacted. The closure of gyms, shutting down of local sports competitions and impact of social-distancing policies will mean that you will soon have to look elsewhere in order to find like-minded people looking to exercise, provide content and guidelines as well as generally still feel connected to an aspect of your life that may be impacted.

Description

The smartphone application will provide users with a sleekly designed UI developed using Java that will be compatible across both Android and iOS device. I'll focus on 5 key areas of the application in this description: Home, Interests, 'Find my Activity' and Feed and Messages.

Home: Upon downloading and installation users will be prompted to enter some basic information that will be used to create their profile and populate their own home page on the application. This would be limited to a short description, location, name and age. It is aimed to be as less intrusive as possible and not require a lot of detailed information - or linking with other social media accounts. This is intended to alleviate any privacy concerns that users may have, and also to streamline the sign-up process and increase the user base.
Interests: Users will be required to select exercises and activities they would be interested in participating in socially with other users. This may include swimming, running, strength training, yoga etc. This selection along with the user's location will allow the app to suggest an activity that directly relates to the user's preferences.
'Find my activity' This module will provide user's the ability to either create an activity they intend to 'lead or host' or search for other activities application user's have already created that are upcoming. Date, time, location, description are all requirements as well as a maximum number of attendees (in order to comply with social distancing measures). Once a user is 'matched' or finds an activity they wish to attend they can apply and leave a short message for the creator of the activity - which will then need to be approved. This means that the original creator has the final say on attendees and can approve/reject user's as they wish. Once accepted the details of the activity will be provided the user both via a message and will no also appear in the user's 'feed' module of the application.
Feed: In order to compete and also provide familiarity with other social media platforms there will be a 'feed' module as part of this application. This will provide users who have connected with other users and activities a platform to share content (e.g. home workouts, fitness information, articles) as well as information on upcoming activities that a user may have agreed to attend. Ideally this content could also incorporate industry-specific articles, news etc. through and RSS-type feed that could pull content. This would be user flexible, with the primary idea to share content and keep users updated with upcoming activities and connected with other users of the application.
Messages: Finally, a module that would provide the ability to send user-to-user messages (provided they have connected in the past). As mentioned, when activities are created and applied for this would be confirmed or rejected via a message. Additionally, messages from the application developer such as new updates, important announcements and new version releases would also be shared here.

Tools & Technologies

The application would be developed using Java and due to the application being mobile-based will result in a low dependency on extensive hardware - basically a compatible laptop/PC with the relevant software such as Eclipse is really all that will be required for the development of the application. The application would be hosted on a cloud server such as Azure App Service, meaning that the cost of hosting the application would be relevant to the number of users as well as any data storage that is required.

Skills Required

Proficiency in coding in Java will be the skill most needed from a software development standpoint. However, depending on the complexity of the application and/or more advanced features that may be required (emails, location services, messaging tools, RSS feeds) additional software skills will probably also need to be investigated and learned. Being a cloud-based solution some of the hardware skills will be automated, however a knowledge of basic database management and prior experience with cloud computing would also be expected. Given that Java is a very popular programming language and cloud computing is surpassing on-premise hardware most providers are extremely competent and seek to reduce complexity for the end-user/administrator as much as possible which means that upskilling for a project like this would be achievable.

Outcome

If the application is successful, this will provide a social media platform for fitness participants that is able to connect people who are interested in similar activities as others without the need for gyms and organised mass-market sporting competitions. Given the current climate and restrictions/closures around these venues and organisations it will it offer users a safe platform to arrange activities and still be able to have an active lifestyle during a time of crisis. Ideally this development will help people with their mental health and provide a healthy outlet whilst still complying with any government recommendations throughout the covid-19 pandemic.
